# Cloud Infra Security Project

## Overview
This project demonstrates AWS Cloud Infrastructure Security best practices, covering IAM, VPC setup, monitoring, encryption, compliance automation, and incident response.

## AWS Services Used
- IAM, VPC, Security Groups, NACLs
- CloudTrail, GuardDuty, AWS Config, Security Hub
- KMS, S3 Encryption
- Lambda, SNS

## How to Use This Repo
1. Clone this repository.
2. Follow steps folder-wise from Step1 to Step6.
3. Each folder has a README for detailed instructions.
4. Run scripts or apply policies as detailed.

## Project Steps Summary
- **Step1-IAM-Policies:** IAM roles, users, and policy JSON examples.
- **Step2-VPC-Architecture:** VPC, subnets, security groups setup scripts.
- **Step3-CloudTrail-GuardDuty:** Monitoring and threat detection automation.
- **Step4-Encryption:** Data encryption with KMS and S3 policies.
- **Step5-Compliance-Automation:** AWS Config and Security Hub enablement.
- **Step6-Incident-Response:** Lambda-based auto remediation and alerts.
